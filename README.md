## 🌏 기술스택 

OS: linux  
DB: neo4j (graph db)  
Language: TypeScript   
Server: Node.js  
Backend: GraphQL, express  
Frontend: React.js, Apollo   
Code Convention: prettier , typescript-eslint  
test: jest  
infra: actions
